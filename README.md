# oh-my-zsh Customizations

## Theme
Set the theme to use for zsh
```
ZSH_THEME="gianu"
```

## Plugins
Set the plugins to use
```
plugins=(git mvn macos sudo vi-mode brew keychain gpg-agent wd)
```
